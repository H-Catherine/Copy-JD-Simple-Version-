# Copy-JD-Simple-Version- - 京东商城简易复刻版

简单的复刻京东商城前端界面😜，使用HTML、CSS和JavaScript实现的一个响应式电商网站原型。

## 项目简介

这是一个模仿京东商城的静态网页项目，实现了京东首页的主要功能模块和页面布局。项目使用原生Web技术栈构建，适合前端初学者学习参考。

## 功能特性

- **完整页面布局**：实现了京东首页的所有主要区域
- **响应式设计**：适配不同屏幕尺寸
- **交互功能**：
  - 轮播图切换
  - 商品分类导航
  - 楼层切换
  - 购物车功能
- **多楼层展示**：包含猜你喜欢、服装鞋包等多个商品楼层
- **底部服务**：购物指南、配送方式、支付方式等服务信息

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6)

## 页面结构

- 顶部通栏
- 主搜索区域
- 导航菜单
- 轮播图展示
- 商品推荐区
- 分类楼层展示
- 底部服务信息
